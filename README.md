# My Finance Application

A full-stack personal finance and budget management web application built with React, Express, and PostgreSQL (with Android Mobile App Sync support).

## 📚 Documentation & Integration Guides

- **[SETUP.md](./SETUP.md)** – Local development & PostgreSQL setup guide.
- **[API_GUIDE.md](./API_GUIDE.md)** – Web API guide for external AI agents, client scripts, and integrations.
- **[PROJECT_BLUEPRINT.md](./PROJECT_BLUEPRINT.md)** – Full system architecture & database schema blueprint.

## 🚀 Getting Started

Please refer to **[SETUP.md](./SETUP.md)** for detailed local setup instructions, including repository cloning, environment configuration, PostgreSQL database initialization, and running in development or production mode.

### Quick Commands

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build and start production
npm run build
npm start
```
